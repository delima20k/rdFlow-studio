📱 Câmera Remota PWA

Transforme seu smartphone em uma câmera profissional wireless para streaming e gravação

🚀 Abrir App 📚 Documentação

✨ Características

📱 PWA Completo

Instalável em Android e iOS como aplicativo nativo. Funciona offline após primeira instalação.

🎥 WebRTC

Transmissão de vídeo em tempo real com latência mínima via conexão peer-to-peer.

📡 Multi-Câmera

Suporte para câmeras frontal e traseira com troca rápida e seleção de dispositivos.

🎨 Responsivo

Interface adaptável para todos os tamanhos de tela, de celulares a smart TVs.

🔐 Seguro

HTTPS obrigatório, permissões explícitas, sem coleta de dados.

⚡ Zero Dependências

HTML, CSS e JavaScript puro. Nenhuma biblioteca externa necessária.

🚀 Como Usar

1️⃣ Teste Local

  • Execute: python -m http.server 8080
  • Ou: bash start.sh (Linux/Mac)
  • Ou: start.bat (Windows)
  • Acesse: http://localhost:8080

2️⃣ Deploy Produção

  • GitHub Pages: Push e ative Pages
  • Vercel: vercel --prod
  • Netlify: Drag & drop no app.netlify.com
  • Firebase: firebase deploy

3️⃣ Conectar ao Backend

  • Edite API_BASE_URL no index.html
  • Ou use: ?api=URL&session=ID
  • Configure CORS no backend
  • Teste a conexão WebRTC
